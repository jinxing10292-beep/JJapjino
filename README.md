# 🎰 온라인 카지노

HTML, CSS, JavaScript로 만든 완전한 온라인 카지노 웹사이트입니다.

## 🎮 포함된 게임

### 🎲 싱글플레이어 게임
- **룰렛**: 빨강/검정, 홀수/짝수에 베팅
- **블랙잭**: 21에 가까운 숫자를 만드는 카드 게임
- **슬롯머신**: 3개의 릴에서 같은 기호 맞추기
- **포커**: 최고의 패를 만드는 카드 게임
- **바카라**: 플레이어 vs 뱅커
- **주사위**: 주사위 합을 맞춰보세요
- **동전던지기**: 앞면 또는 뒷면
- **가위바위보**: 컴퓨터와 가위바위보 도박
- **달팽이 레이싱**: 4마리 중 우승할 달팽이 선택
- **행운의 바퀴**: 큰 상금을 노려보세요
- **복권**: 번호를 맞춰 잭팟을 노리세요
- **크래시**: 언제 멈출지 결정하세요

### ⚽ 스포츠 토토
- 가상의 축구, 농구, 테니스, 배구 경기
- 실시간 배당률 시스템
- 15개 유명 팀들의 랜덤 매치업

### 🌐 PvP 멀티플레이어 게임
- **PvP 블랙잭**: 다른 플레이어와 실시간 대전
- **PvP 포커**: 실시간 포커 대결
- **PvP 가위바위보**: 빠른 대전 모드

### 🏆 랭킹 시스템
- **잔액 랭킹**: 가장 많은 돈을 가진 플레이어
- **승수 랭킹**: 가장 많이 이긴 플레이어
- **게임 수 랭킹**: 가장 많이 플레이한 플레이어
- **🔮 랭킹 예측 베팅**: 10분 후 랭킹을 예측하고 베팅 (정확히 맞추면 100배!)

## 🚀 배포 방법

### GitHub Pages (기본)
1. 이 저장소를 GitHub에 업로드
2. Settings > Pages에서 Source를 "Deploy from a branch" 선택
3. Branch를 "main" 선택하고 Save
4. 몇 분 후 `https://[사용자명].github.io/[저장소명]`에서 접속 가능

### Supabase 온라인 기능 설정 (선택사항)

#### 1. Supabase 프로젝트 생성
1. [Supabase](https://supabase.com)에 가입
2. 새 프로젝트 생성
3. 프로젝트 URL과 anon key 복사

#### 2. 데이터베이스 설정
1. Supabase 대시보드 > SQL Editor
2. `supabase-schema.sql` 파일의 내용을 복사하여 실행
3. 테이블과 정책이 생성됨을 확인

#### 3. JavaScript 설정 수정
`script.js` 파일에서 다음 부분을 수정:
```javascript
const SUPABASE_URL = 'https://your-project.supabase.co'; // 실제 URL로 변경
const SUPABASE_ANON_KEY = 'your-anon-key'; // 실제 키로 변경
```

#### 4. Supabase 클라이언트 초기화
```javascript
supabase = window.supabase.createClient(SUPABASE_URL, SUPABASE_ANON_KEY);
```

## 🎯 게임 규칙

### 🔮 랭킹 예측 베팅 (신규!)
- 10분마다 랭킹이 업데이트됩니다
- 특정 플레이어가 몇 위가 될지 예측하고 베팅
- **정확히 맞추면 베팅 금액의 100배 획득!**
- 한 플레이어당 하나의 예측만 가능
- 실시간 카운트다운으로 남은 시간 확인

### 룰렛
- 베팅 금액을 입력하고 색깔이나 홀짝을 선택
- 맞추면 베팅 금액의 2배를 획득

### 블랙잭
- 21에 가까운 숫자를 만드는 것이 목표
- 21을 초과하면 버스트로 패배
- 딜러보다 높은 숫자를 만들면 승리

### 스포츠 토토
- 가상 경기에서 승/무/패 선택
- 배당률에 따라 당첨금 결정
- 경기는 랜덤하게 새로 생성됨

### PvP 게임
- 닉네임을 입력하고 매칭 시작
- 실시간으로 다른 플레이어와 대전
- 현재는 시뮬레이션 모드 (Supabase 연결 시 실제 멀티플레이어)
