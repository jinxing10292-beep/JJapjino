# 🎰 온라인 카지노

HTML, CSS, JavaScript로 만든 간단한 카지노 게임 웹사이트입니다.

## 🎮 포함된 게임

- **룰렛**: 빨강/검정, 홀수/짝수에 베팅
- **블랙잭**: 21에 가까운 숫자를 만드는 카드 게임
- **슬롯머신**: 3개의 릴에서 같은 기호 맞추기

## 🚀 배포 방법

1. 이 저장소를 GitHub에 업로드
2. Settings > Pages에서 Source를 "Deploy from a branch" 선택
3. Branch를 "main" 선택하고 Save
4. 몇 분 후 `https://[사용자명].github.io/[저장소명]`에서 접속 가능

## 🎯 게임 규칙

### 룰렛
- 베팅 금액을 입력하고 색깔이나 홀짝을 선택
- 맞추면 베팅 금액의 2배를 획득

### 블랙잭
- 21에 가까운 숫자를 만드는 것이 목표
- 21을 초과하면 버스트로 패배
- 딜러보다 높은 숫자를 만들면 승리

### 슬롯머신
- 3개 릴이 모두 같으면 대박
- 2개가 같으면 소액 당첨
- 다이아몬드(💎) 3개: 10배, 7️⃣ 3개: 8배, 기타 3개: 5배
